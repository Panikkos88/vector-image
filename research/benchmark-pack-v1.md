# Benchmark Pack v1 - 2026-06-27

## Purpose

Create a generated, license-safe benchmark pack so the router and future engine changes are tested on more than BOC/NS CAR. The images are generated by `tools/make-benchmark-logos.js` and live in `app/assets/benchmarks/`.

The benchmark route is hidden: `?asset=assets/benchmarks/<file>.png`. It is for testing only and does not change the user-facing upload flow.

Settings under test:

- Auto router
- Medium detail
- Smooth anti-aliasing
- Balanced sub-pixel edges
- Balanced curve optimizer
- Preserve glows/shadows
- Background Detach Off

## Files

- Generated inputs: `app/assets/benchmarks/*.png`
- Manifest: `app/assets/benchmarks/manifest.json`

## Findings

1. The automatic router is broadly sane on this pack: simple flat/fine/glow/transparent logos route to Palette; outline and metallic/gradient cases route to Region.
2. Palette is close on simple flat badges, but VM still wins with fewer nodes and bytes.
3. Fine text is the first sharp Palette regression target: ours has 179 paths and 4.62% edge RMSE, while VM has 61 paths and 2.26%.
4. Glow/shadow color modeling is not solved by our current Palette path. VM gets dark-glow MAE near 0.09%; ours is 2.04%.
5. Region is the wrong long-term answer for outline/metal logos in its current form. The router routes there for understandable reasons, but VM is dramatically better on both edge error and hot pixels.
6. The next useful work should optimize against this pack, not only BOC. BOC is now a solved edge-bar sample; this pack exposes the next ceiling.

## Next Technical Targets

1. Add a VM-reference measurement harness so `research/vm-benchmarks/*.svg` can be scored from one command or UI action.
2. Fine-text Palette pass: preserve thin strokes while reducing path count, likely by improving palette-connected component merging and contour simplification around text-like components.
3. Glow/shadow pass: stop forcing soft effects into flat fills; test explicit opacity/blur/gradient layers against `bench-dark-glow`.
4. Router pass: add a rule that rejects Region for clean outline logos when the palette ladder is only barely over the current `k <= 4` guard, because `bench-outline-shield` shows Region can be much worse.
5. Transparent VM pass: capture VM with transparent/removal settings or update the metric harness to compare on a chosen compositing background.
